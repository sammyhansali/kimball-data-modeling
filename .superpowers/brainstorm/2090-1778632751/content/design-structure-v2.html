└── main.py
Why this structure
Self-contained chapters

Each chapter directory is a complete, runnable dbt project. You can cd ch03_retail && dbt run independently.

Shared Python env

One pyproject.toml at the root installs dbt-duckdb once. All chapter projects share it.

Each chapter its own DuckDB file

dbt-duckdb will write to ch03_retail/dev.duckdb by default — isolated per chapter, nothing shared.
